## 🚀 What ACI is

ACI stands for Agent-Computer Interface. It gives software a clear, structured way to work with agents and automation tools. In plain terms, it helps a computer app expose its controls in a format that other tools can understand.

With ACI, you can use supported apps in a more direct way. Instead of relying on screenshots or guesswork, the app can present a stable interface for actions, text, and navigation. This makes desktop automation easier to manage on Windows.

## 🧭 How ACI works

ACI uses a structured interface instead of screen snapshots. That means other tools can read the app in a more stable way.

This is useful for:

- desktop automation
- browser automation
- computer use tasks
- agent workflows
- tool-based input and output
- repeated tasks that need consistent steps

ACI is built for systems that need to interact with software in a clear, predictable way.

## 🔧 Basic use

To get started, try this simple flow:

1. Open ACI.
2. Connect the app you want to control.
3. Choose an action from the list.
4. Send the action to the app.
5. Check the result.
6. Repeat the next step as needed.

If you are using ACI with an agent tool, the agent can use the interface to decide what to do next. This helps reduce manual clicking and makes each step easier to track.

## 🧰 If something does not open

If ACI does not start, try these steps:

1. Right-click the file and choose **Run as administrator**.
2. Check that the file finished downloading.
3. Make sure Windows did not place it in quarantine.
4. Restart your computer and try again.
5. Download the latest release again from the releases page.
6. Close other automation tools if they are using the same app.
7. Try launching from the installed folder instead of a shortcut.

If the window opens but nothing responds, check whether the target app is already open and ready for input.

## 🛠️ Troubleshooting

If ACI runs, but your workflow fails, check these points:

- The target app is open.
- The correct window is active.
- The app has not frozen.
- Your mouse and keyboard are free for use.
- No other app is blocking input.
- The latest release is installed.
- You are using the right version for Windows.

If text input looks wrong, try typing into Notepad first to confirm your keyboard works.

## 📌 Tips for smooth use
